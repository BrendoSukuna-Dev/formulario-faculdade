## 📋 Sobre o projeto
<p>Este projeto consiste em um formulário web completo e estilizado, desenvolvido para capturar informações de estudantes de tecnologia. Ele foi construído com foco em semântica HTML5,
  validação de campos e design personalizado utilizando CSS3.</p>


## ✦ Principais funcionalidades:

<p>➥ Captura de Dados Abrangente: Coleta de nome, sobrenome, idade, e-mail e curso.</p>
<p>➥ Seleção de Perfil Profissional: Uso de botões de rádio para definir o foco de atuação (Front-end, Back-end, Fullstack).</p>
<p>➥ Mapeamento de Habilidades: Checkboxes para seleção múltipla de tecnologias (HTML, CSS, JS, PHP, C#, Python, Java).</p>
<p>➥ Integração com Backend: Configurado para envio de dados via Formspree, facilitando o recebimento das respostas por e-mail.</p>
<p>➥ Interface Temática: Design inspirado na cultura pop (tema Sukuna), com uso de imagens de fundo, fontes sans-serif e uma paleta de cores roxa consistente.</p>
